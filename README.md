# Juegos

Proyecto Django para generar y exportar **crucigramas**, **sopas de letras**, **sudokus**, **dibujos para colorear**, **cuadernos de caligrafía** y **mandalas**.  
Incluye un **Lexicon** para gestionar palabras y categorías.
